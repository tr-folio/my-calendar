Push for First Commit:
git init
git add .
git commit -m "first commit"
git branch -M main
git remote add origin https://github.com/tr-folio/git-necessary-codes.git
git push -u origin main

Push to the same repository from second commit:
git add .
git commit -m "commit message"
git push

Push to an existing different repository:
git remote add origin https://github.com/tr-folio/git-necessary-codes.git
git branch -M main
git push -u origin main
